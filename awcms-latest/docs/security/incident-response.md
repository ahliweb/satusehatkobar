# Incident Response

## Minimum Process

1. Detect and triage the incident.
2. Contain access or data exposure.
3. Preserve logs and evidence.
4. Assess customer, operational, and regulatory impact.
5. Recover services safely.
6. Document lessons learned and remediation.

## Typical Inputs

- audit logs
- deployment history
- Cloudflare event data
- database and storage access records
